Este es un sencillo proyecto realizado íntegramente en Angular v16.0.0 y potenciado con la biblioteca de componentes PrimeNG v16.0.0-rc.1. Lo que hace especial a este proyecto es que es puramente frontend, lo que significa que no depende de ninguna API para su funcionamiento. Toda la información, imágenes y contenido que encontrarás aquí fueron investigados y agregados manualmente por mí. Cada detalle, desde la elección de imágenes hasta la disposición de la información, ha sido cuidadosamente considerado y diseñado para brindarte una experiencia atractiva y efectiva.
